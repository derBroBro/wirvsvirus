Öffentliche Daten, dass is unser Ziel

Unsere oberstes Ziel ist eine öffentliche Platform zum austausch von anonymen Gesundeheitsdaten zu schaffen.

Bitte beachte
Die Daten sind nicht qualifiziert, dass heist jeder kann seine subjektive Empfindung kund tun.
Beachten zudem, dass die Daten mehrfach eine Probe abgegeben werden kann. Diese findet sich dann unter gleicher person_id auch im Datensatz wirder. Es ist dir überlassen wir du damit umgehst.
Wir arbeiten stark daran, Script und Bots zu unterbinden, jedoch können wir dies nicht ausschließen.


Bitte gehen daher mit deinen Schlüssen und Ergebnissen verantwortungsbewusst um.

Die Daten setzten sich wie folgt zusammen:

Feld Beschreibung
sender_id Der Hash des Absenders (md5 + fingerprintjs2). Ein Absende kann auch Proben für Aliase abgeben. Diese haben jedoch immer die gleiche sender_id.
person_id Der Hash des Absenders + Alias (md5 + fingerprintjs2). Diese ID sollte bei gleichem Absender und Alias gleich bleiben.
sample_creation Wann (utc) wurde die Probe abgegeben.
target_kind Wurde die Proble für einen selbst (self) oder jemanden anderen (other) abgegeben.
age Altersgruppe der Person.
location Standort der Person.
symptoms_duration Wie lange hat die Person schon die Symptome. 7 = eine Woche oder mehr.
symptoms_calculated_start Errechneter Start der Symptome (Abgabedatum - Tage).
symptoms_throat
symptoms_headache
symptoms_limb
symptoms_cough
symptoms_fever		 Stärke der Symptome 0 = Keine, 1 = Leicht, 2 = Mittel, 3 = Stark
covid19_tested Wurde eine Test durchgeführt? 0 = Nein, 1 = Ja
covid19_test_result Ergebniss des Tests. 0 = negativ, 1 = positiv
covid19_test_duration Wie lange (Tage) musste auf das Ergebniss gewartet werden. 7 = eine Woche oder mehr.
